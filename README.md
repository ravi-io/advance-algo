## 📌 Project Overview

This repository contains 10 practical Java programs focused on:

- File Handling
- Sorting and Searching
- Matrix Operations
- Exception Handling
- Multithreading
- Collections Framework
- JDBC Connectivity
- CSV Processing
- Serialization and Deserialization

## ▶️ How to Run

### 1. Compile all programs

```bash
javac *.java
```

### 2. Run any program

```bash
java FileHandling
java SortingAlgorithms
java SearchingTechniques
java MatrixOperations
java ExceptionHandlingDemo
java MultithreadingDemo
java JavaCollectionsDemo
java JDBCConnectivityDemo
java CSVProcessingDemo
java ObjectSerializationDemo
```

## 🧠 Notes

- The code is written in a clean, beginner-friendly style for lab practice.
- Time and space complexity are mentioned where applicable.
- JDBC requires a running MySQL server and valid credentials.
- CSV processing expects a file named `students.csv` in the project folder.
